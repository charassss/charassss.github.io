开始学习kali

发表于 2022-03-19 分类于 kali

#认识kali

Kali Linux是一个基于Debian的开源Linux发行版,面向各种信息安全任务,如渗透测试、安全研究、计算机取证和逆向工程.

Kali Linux is an open-source, Debian-based Linux distribution geared towards various information security tasks, such as Penetration Testing, Security Research, Computer Forensics and Reverse Engineering.

Kali官网

使用虚拟机部署Kali Linux

在您的虚拟机上部署Kali Linux，可以使用以下方法:

使用Kali Linux ISO文件，将其安装到虚拟机上 步骤一

下载 Kali Linux ISO文件 (64位) [推荐]
下载 Kali Linux ISO文件 (32位)
下载 Kali Linux ISO文件 (ARM)

下载后,打开您的虚拟机控制软件,并使用ISO文件安装Kali Linux.

本文使用VMware Workstation作为示范

第一步

  • 新建一个虚拟机

  • 选择 典型(推荐)
  • 点击 下一步

  • 选择 稍后安装操作系统
  • 点击 下一步

    • 虚拟机操作系统选择 Linux
    • 版本选择 Debian 9.x 64位 (若您的ISO镜像为32位,请选择Debian 9.x 32位)
  • 点击 下一步

使用Kali Linux ISO文件，将其安装到虚拟机上 步骤二
    • 虚拟机名称随意,建议有明显表示名
    • 虚拟机安装位置请选择一个大容量磁盘,具体位置请根据自己的需要选择,不建议路径中包含中文以及特殊字符以及空格,不建议虚拟机安装在系统盘上
  • 点击 下一步

    • 最大磁盘大小建议在 40GB 以上
    • 选择 将虚拟机存储为单个文件
  • 点击 下一步

  • 检查信息
  • 单击完成

使用Kali Linux ISO文件，将其安装到虚拟机上 步骤三

第二步

  • 点击 开启此虚拟机

  • 如图,选择图中第一个选项 Graphic Install
  • 点击 continue

  • 选择 中文(简体)
  • 点击 continue

  • 点击两次 继续
  • 主机名填什么都可
  • 点击 继续

  • 域名保持不变
  • 点击 继续

  • 用户名填自己想要在登录时输入的名字
  • 点击 继续

  • 全名保持不变即可
  • 点击 继续

  • 设置密码,请牢记密码,以后会作为登录时输入的密码使用
  • 点击 继续

  • 选择 向导-使用整个磁盘 即可
  • 点击 继续

  • 应该只会出现一个选项,若出现多个,请选择容量最大的磁盘
  • 点击 继续

  • 选择 将所有文件放在同一个分区(推荐新手使用) (若您希望自定义分区,请选择其他选项)
  • 点击 继续

  • 选择 结束分区设定并将修改写入磁盘
  • 点击 继续

  • 选择 是
  • 点击 继续

  • 不加改动,点击继续即可

  • 选择 是
  • 点击 继续

  • 选择/dev/sda
  • 点击 继续

  • 点击 继续

收尾工作
  • 稍作等候,输入用户名与密码即可进入新系统


恭喜您,安装完成
